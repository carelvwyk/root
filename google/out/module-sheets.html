Methods

(inner) insertRow()

insertRow inserts "data" after the last non-empty row on the sheet. Only values for keys that match column titles will be added in those columns, the rest of the values will be ignored.

(inner) loadTable(sheetName) → {TableInfo}

loadTable assumes spreadsheet data is structured as a table with one header row and 0 or more data rows. It returns an associative array of column indexes by name (lowercased) and the rest of the table data (excluding the header row).
Parameters:
Name Type Description
sheetName string the name of the sheet to load
